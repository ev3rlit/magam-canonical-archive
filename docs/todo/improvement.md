백로그 개선 사항

- 스티커, 와시테이프, 스티키, 마인드맵등 각 요소들에 대해서 기본 zorder를 정리하고 스티커 및 와시테이프는 항상 최상위에 있도록 수정
- 노드를 눌러서 복사시 id를 추출하도록 수정
- 노드 우클릭해서 오브젝트 복사 기능 추가하기
- magam 스킬을 좀더 개선해서 사용자에게 의도를 물어보도록 개선
- 내부 컨텐츠 양에 따른 동적 크기 조절 및 여백 제거 개선
- 시퀀스 다이어그램을 마인드맵 노드로 추가할시, 겹치는 문제. (크기 계산이 정확하지 않은 가봄)
- 절대 좌표 기반 대신 상대 기반 앵커 기능을 개선 해야함.
- 최우선 양방향 편집 기능의 필요성. 해보면 실제로는 내가 직접 추가해서 하면 빠르게 할수있을때가 있음.
